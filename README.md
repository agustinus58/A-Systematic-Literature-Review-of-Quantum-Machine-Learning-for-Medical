# A-Systematic-Literature-Review-of-Quantum-Machine-Learning-for-Medical
To further strengthen the methodological rigor, a quality assessment was conducted to evaluate the scientific validity of each included study. The quality assessment criteria used in this study are presented in Table 3, including: (QA1) clarity of research objectives, (QA2) methodological transparency, (QA3) dataset and experimental setup, (QA4) validity of evaluation results, and (QA5) relevance to the medical Quantum Machine Learning (QML) domain. Each criterion was scored using a scale of 1 (Yes), 0.5 (Partially), and 0 (No).
